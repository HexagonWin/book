ㄷ. 부트로더

 

1. 부트로더란 무엇인가요?

 - 간혹 가다, 베가레이서 기종을 루팅하실때, 부트로더 언락이라는 과정을 거쳐가야 합니다. 그 때, fastboot mode 로 진입되는데 그게 바로 부트로더 입니다.

   Windows 의 Bootloader 도 마찬가지 이지만, OS 실행을 위한 환경을 설정해 주는 '중요한 역할을 해 줍니다.

 

 

2. 부트로더가 날라가면 어떻게 되나요?

 - 휴대폰의 장례식을 치러줘야 합니다.(공사장 직행) 부팅이 되지 않습니다. (하드브릭)

 

 

3. 부트로더를 복구할 수 있는 방법은 없나요?

 - HBR Project (HardBrick Recovery Project, 하드브릭 복구 프로젝트)가 SDA에서 ANT팀을 시발으로 하여 많은 연구가 이루어졌고 이를 통해 BINX Project 게시판에 많은 관련 자료가 있으니 찾아보시길 바랍니다
